The Game of Nim

아래 여러개의 동전 더미가 있다. 각 플레이어는 자신의 차례가 오면 한 더미를 골라 최소 1개 이상의 동전을 가져가야 한다. 만약 동전을 가져갈 수 없으면 게임에서 패배한다.

따라서 마지막 동전을 가져간 사람이 승리한다.

아래 New Game 버튼을 눌러 게임을 시작하거나 Option 버튼을 통해 게임 규칙을 직접 정해보세요.
참고로 바로 New Game 버튼을 누르시거나 클래식 모드를 설정하셨으면 컴퓨터가 먼저 시작합니다.
이외에도 랜덤하게 생성되었을 경우 드물게 컴퓨터가 먼저 시작할 수도 있습니다.
