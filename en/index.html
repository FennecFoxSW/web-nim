The Game of Nim

There are several piles of coins below. Each player must pick a pile and take at least one coin when it is player's turn.
If player is unable to take he loses.

So the one who takes the last coin is the winner.

Click the 'New Game' button below to start the game, or click the 'Option' button to set game rules.
Game Options

Set rules of the game.

Classic Mode
(It means a game in which 1, 3, 5, and 7 coins are placed in 4 piles each. If classic mode is on, the computer starts first.)

Set the number of piles of coins.

Keyboard: New Game: N | Pass turn: Enter, Space | Options: Esc | First pile: 1, D | Second pile: 2, F | Third pile: 3, J | Forth pile: 4, K
